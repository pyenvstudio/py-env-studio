ClickPy Download Badges

Show your PyPI download statistics with stylish badges

How to Use These Badges

  1. Copy the Markdown code for the badge you want to use
  2. Paste it into your README.md file
  3. The badge will automatically show the current download count
  4. Replace "clickpy" with your actual package name if needed
